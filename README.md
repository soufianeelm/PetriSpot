## how to install :

```
git clone -b invariant https://github.com/soufianeelm/PetriSpot.git

cd PetriSpot

./build.sh
```


## how to use :

```
cd Petri/src

./petri [model.pnml] [flags]
```


### [model.pnml] : 

path of model file in format .pnml


### [flags] :

--findDeadlock

--Psemiflows || --Pflows

--Tsemiflows || --Tflows
